Objective: Create a custom structural directive that shows or hides an element based on a condition.

1. Create a directive called appShowIf.

2. The directive should take a boolean value and show the element if the value is true, otherwise hide it.

Example Usage:
<div *appShowIf="true">This is visible</div>
<div *appShowIf="false">This is hidden</div>
Expected Result:
<div class="card" *appShowIf="condition">...</div>
